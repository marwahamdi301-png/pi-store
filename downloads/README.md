# نظام جمال الماستر | Jamal Master System

نظام برمجي متطور مبني بلغة **Python**، مصمم لإدارة الحسابات وتحليل بيانات السوق بشكل آلي ومحترف.

## 🚀 المميزات الرئيسية
* **تحليل تلقائي:** معالجة دقيقة لبيانات السوق وتوليد نتائج فورية.
* **نظام تقارير ذكي:** يتم إنشاء ملفات تقارير زمنية منظمة في مجلد `Reports`.
* **هيكلية احترافية:** كود نظيف ومنظم مهيأ للتطوير المستقبلي والربط مع أدوات التداول.
* **التوافق:** يعمل بسلاسة على بيئة **Termux** للأجهزة المحمولة.

## 📁 هيكلة المشروع
* `master_jamal.py`: الملف الرئيسي لتشغيل النظام.
* `Reports/`: المجلد المخصص لحفظ التقارير المستخرجة.
* `README.md`: ملف التعريف بالمشروع.
